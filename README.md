# About astro-notebooks
This is a computational exploration space dedicated to physics, mathematics and astronomy.

This branch serves as a hybrid between:

-Academic study notebooks
-Technical demonstrations of scientific Python
-Narrative-driven computational explorations

Each notebook is designed not only to implement equations, but to investigate physical systems through numerical modeling, simulation and visualization.

---

## Scope

The notebooks cover topics such as:

-Mathematical methods for physics
-Classical mechanics and gravitation
-Electromagnetism and wave phenomena
-Astrophysical modeling
-Cosmological frameworks
-Numerical methods and simulations

---

## Approach
Most notebooks will follow a structured flow:

-Theoretical context
-Mathematical formulation
-Computational implementation
-Simulation and visualization
-Critical analysis and extensions

The goal is not only to make codes, but to understand phenomenas and theories, and make it more understandable.
